Byakuya MD

A multi-device WhatsApp bot for otaku culture with 20+ anime and manga-themed plugins

Node.js Multi-Device 20+ Plugins Baileys

Features

Multi-device Support

Works with QR code and pairing code authentication for seamless connection.

20+ Plugins

Comprehensive collection of anime and manga-themed functionality.

Anti-spam Protection

Built-in rate limiting and cooldown system to prevent abuse.

Admin Controls

Special commands for bot administrators to manage the bot effectively.

Cross-platform

Deploy on Heroku, Railway, VPS, or Docker with easy configuration.

Fast & Reliable

Built on Baileys library for optimal performance and stability.

Installation

Follow these steps to set up Byakuya MD on your system:

Prerequisites

  • Node.js 16 or higher
  • npm or yarn
  • WhatsApp account
  • FFmpeg (for audio processing)

Step-by-Step Guide

# Clone the repository git clone https://github.com/your-username/byakuya-md.git cd byakuya-md # Install dependencies npm install # Set up environment variables cp .env.example .env # Edit .env with your configuration # Run the bot npm start # Scan the QR code with WhatsApp to link your device

Environment Variables

Variable Description Required
ADMIN_NUMBERS Comma-separated admin numbers with country code Yes
MAL_CLIENT_ID MyAnimeList API client ID No
ANILIST_TOKEN AniList API token No
GIPHY_API_KEY Giphy API key for GIFs No
TENOR_API_KEY Tenor API key for GIFs No
LOG_LEVEL Logging level (info, debug, error) No

Usage

Once the bot is running, you can use the following commands:

.help Show all available commands
.quote Get a random anime quote
.waifu Get a random waifu image
.manga naruto Search for Naruto manga
.schedule monday Show Monday's anime schedule
.battle @friend Start a waifu battle with a friend
/restart Restart the bot (admin only)
/reload all Reload all plugins (admin only)
